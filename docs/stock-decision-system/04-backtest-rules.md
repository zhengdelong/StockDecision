# 回测规则

## 回测目标

回测用于验证策略扣除成本后是否有盈利可能。没有通过回测的策略不能进入实盘辅助，只能作为学习观察。

回测不是证明未来一定赚钱，而是排除明显无效或过拟合的规则。

## 回测周期

默认回测最近 5 年。

额外分段：

- 最近 1 年：观察当前市场适应性。
- 最近 3 年：观察中期稳定性。
- 最近 5 年：观察跨周期表现。

## 初始资金

```text
初始资金 = 20000 元
```

## 交易成本

默认成本：

| 成本项 | 默认值 |
|---|---:|
| 买入佣金 | max(成交额 * 0.03%, 5 元) |
| 卖出佣金 | max(成交额 * 0.03%, 5 元) |
| 卖出印花税 | 成交额 * 0.05% |
| 买入滑点 | 成交额 * 0.1% |
| 卖出滑点 | 成交额 * 0.1% |

所有成本必须从收益中扣除。若用户实际券商费率不同，后续做成配置项。

## 买入价格

信号在收盘后生成，因此不能使用当天收盘价作为实际买入价。

默认买入规则：

```text
信号日 = T
买入日 = T + 1
买入价 = T + 1 开盘价 * (1 + 买入滑点)
```

若高开超过策略阈值：

- 强势突破高开超过 3%，跳过。
- MA20 回踩高开超过 2%，跳过。

若开盘后价格已经跌破放弃条件，跳过。

## 卖出价格

卖出信号在收盘后确认：

```text
卖出信号日 = T
卖出日 = T + 1
卖出价 = T + 1 开盘价 * (1 - 卖出滑点)
```

止损、跌破 MA20、回撤 10%、止盈减仓都按该规则执行。

## 仓位规则

回测必须严格使用 2 万本金仓位规则：

- 最多同时持有 2 只股票。
- 单票金额 8000 到 10000 元。
- 买入股数按 100 股向下取整。
- 向下取整后买入金额低于 6000 元，跳过。
- 单笔最大亏损目标为 400 元。

## 买入策略回测

回测两个策略：

1. 强势突破。
2. 强势股回踩 MA20。

同时回测组合策略：

```text
组合策略 = 强势突破 + 强势股回踩 MA20
```

每笔交易必须记录策略来源，方便判断哪个策略真正贡献收益。

## 卖出规则回测

必须模拟：

- 初始止损。
- 浮盈 8% 后止损上移到成本价。
- 浮盈 15% 卖出 30%。
- 浮盈 25% 卖出 50%。
- 剩余仓位跌破 MA20 清仓。
- 从最高收盘价回撤 10% 清仓。

减仓后剩余股数也必须保持 100 股整数倍。如果不足 100 股，直接清仓。

## 输出指标

每次回测必须输出：

| 指标 | 说明 |
|---|---|
| 总收益 | 期末资金 / 初始资金 - 1 |
| 年化收益 | 折算后的年化表现 |
| 最大回撤 | 资金曲线最大跌幅 |
| 胜率 | 盈利交易数 / 总交易数 |
| 平均盈利 | 盈利交易平均收益 |
| 平均亏损 | 亏损交易平均亏损 |
| 盈亏比 | 平均盈利 / 平均亏损绝对值 |
| 交易次数 | 完整买卖交易数量 |
| 年均交易次数 | 每年平均交易数量 |
| 连续亏损次数 | 最大连续亏损笔数 |
| 手续费合计 | 所有佣金、印花税、滑点合计 |
| 策略贡献 | 突破和回踩分别贡献多少收益 |

## 验收标准

最低要求：

| 指标 | 要求 |
|---|---:|
| 扣成本后收益 | 大于 0 |
| 对比基准 | 跑赢沪深300或中证500 |
| 平均盈利 / 平均亏损 | >= 2 |
| 胜率 | >= 40% |
| 最大回撤 | <= 20% |
| 年交易次数 | 12 到 50 次 |
| 连续亏损次数 | <= 5 次 |

如果任一核心指标不达标：

- 不展示“可买”信号。
- 只展示“策略观察”。
- 在学习页提示策略仍未通过回测。

## 防止过拟合

v1 不做自动参数优化。所有默认参数先固定：

- MA20、MA60、MA120。
- ATR14。
- 20 日新高。
- 成交额放大 1.5 倍。
- 高开限制 2% 到 3%。
- 单笔亏损 400 元。
- 盈亏比最低 2。

后续如果调整参数，必须保存为新的策略版本，不能覆盖旧版本回测结果。

## 回测失败解释

系统必须能解释回测失败原因：

- 胜率低：买点质量不足。
- 盈亏比低：止盈太早或止损太松。
- 回撤大：仓位太重或市场过滤不足。
- 交易次数少：过滤过严。
- 扣成本后亏损：交易太频繁或单笔金额太小。
- 某策略拖累收益：拆分突破和回踩分别查看。
