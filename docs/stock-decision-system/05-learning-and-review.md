# 学习与复盘

## 学习目标

系统不只输出股票，还要帮助用户理解：

- 为什么这只股票强。
- 为什么现在能买或不能买。
- 错了在哪里止损。
- 对了怎么止盈。
- 每次交易验证了哪个投资知识点。

## 六个月学习路线

| 月份 | 主题 | 应掌握内容 |
|---|---|---|
| 第 1 月 | 趋势和均线 | MA20、MA60、趋势、回踩、突破 |
| 第 2 月 | 成交和资金 | 成交额、换手率、放量、缩量、长上影 |
| 第 3 月 | 风险和仓位 | 止损、最大亏损、盈亏比、回撤 |
| 第 4 月 | 市场和板块 | 大盘环境、行业强度、相对强度 |
| 第 5 月 | 基本面排雷 | PE、PB、ROE、营收、净利润 |
| 第 6 月 | 回测和复盘 | 胜率、盈亏比、交易成本、策略迭代 |

## 信号解释模板

每条信号必须包含以下解释：

```text
股票：{代码} {名称}
信号类型：{强势突破 / MA20 回踩 / 等突破 / 等回踩}

为什么强：
1. {相对强度解释}
2. {趋势结构解释}
3. {成交额解释}
4. {板块强度解释}

为什么现在可以买或不能买：
1. {买点条件是否满足}
2. {高开限制}
3. {是否距离 MA20 太远}

风险在哪里：
1. {止损价}
2. {最大亏损}
3. {主要风险项}

预期收益：
1. {目标价}
2. {预期收益金额}
3. {盈亏比}

本次学习点：
{本次交易最应该学习的一个概念}
```

## 买入前检查

每笔交易前必须回答：

```text
1. 市场环境允许开仓吗？
2. 这只股票属于强势行业吗？
3. 这只股票是否比大盘和行业更强？
4. 买入价是多少？
5. 高开超过多少放弃？
6. 止损价是多少？
7. 最多亏多少钱？
8. 目标价是多少？
9. 预期赚多少钱？
10. 盈亏比是否大于等于 2？
```

任一问题无法回答，不允许生成买入计划。

## 常见错误标签

复盘时必须选择至少一个标签：

- 追高。
- 不按止损。
- 过早止盈。
- 忽略大盘环境。
- 忽略板块弱势。
- 买入时盈亏比不足。
- 数据缺失仍交易。
- 情绪化交易。
- 仓位过重。
- 策略外交易。

## 学习页内容结构

学习页按概念组织：

- 均线：MA20、MA60、MA120。
- 趋势：突破、回踩、趋势排列。
- 量价：成交额、换手率、放量、长上影。
- 风险：ATR14、最大回撤、止损、仓位。
- 收益：目标价、盈亏比、止盈。
- 基本面：PE、PB、ROE、净利润同比。

每个概念页面包含：

```text
1. 白话解释
2. 计算方式
3. 系统怎么使用
4. 正确例子
5. 错误例子
6. 新手误区
```

## 学习进度

系统记录：

- 已阅读指标数量。
- 已完成复盘数量。
- 策略内交易数量。
- 策略外交易数量。
- 是否连续执行止损。
- 是否连续遵守高开放弃规则。

学习目标不是看完文档，而是减少重复错误。
